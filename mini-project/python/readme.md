# Weather Data Analyzer & Forecast Dashboard - Python

## Overview

This project is a Python-based weather analysis system that collects real-time weather data, stores it locally, and provides insights such as trends and basic future predictions.

## Features

* Fetch real-time weather data using OpenWeatherMap API
* Store historical data using SQLite
* Visualize temperature trends
* Predict future temperature using Linear Regression
* Simple CLI-based dashboard

## Tech Stack

* Python
* SQLite
* Pandas
* Matplotlib
* Scikit-learn

## Setup Instructions

1. Clone the repository

```
git clone <your-repo-url>
cd weather-project
```

2. Install dependencies

```
pip install requests pandas matplotlib scikit-learn
```

3. Get API Key

* Sign up at OpenWeatherMap
* Replace `YOUR_API_KEY` in code

4. Run the project

```
python main.py
```

## Usage

* Option 1: Fetch latest weather data
* Option 2: View temperature trends
* Option 3: Predict tomorrow's temperature

## Notes
* Better predictions require more data points
