# Library Management System (Java)

A simple console-based Java application to manage library operations like adding books, issuing, returning, and tracking fines.

## Features
- Add and manage books
- Register users
- Issue and return books
- Automatic due date (7 days)
- Fine calculation (₹5/day late)
- Search books by title or author

## Tech Stack
- Core Java (OOP)
- Collections Framework
- Java Time API

## How to Run
1. Compile:
   javac *.java

2. Run:
   java Main
